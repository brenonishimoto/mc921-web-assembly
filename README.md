# mc921-web-assembly
Projeto de demonstração de webassembly para um seminário de MC921
- Compilação para WebAssembly
- Consumo das funções C no JavaScript via `cwrap`

## Pré-requisitos
- Emscripten SDK instalado e ativado na sessão do terminal.

## Build
### Linux (Bash)
chmod +x build.sh
./build.sh
